# Трекер привычек
Проект по созданию трекера для отслеживания полезных привычек и формирования новых жизненных установок. Он помогает систематизировать повседневные дела, фиксируя выполнение желаемых действий и мотивируя пользователей продолжать и достигать поставленных целей постепенно и эффективно.

## Описание

Трекер привычек — это инструмент, который помогает пользователям отслеживать свои привычки и достижения. Он позволяет:
- Создавать и отслеживать привычки
- Устанавливать цели и отслеживать прогресс
- Анализировать данные и получать отчёты

## Установка

Для установки трекера привычек выполните следующие шаги:

1. Клонируйте репозиторий:
   ```
     git clone https://github.com/AlisaKarpova/Habit-Tracker.git
   ```
   
2. Установите зависимости:
   ```
   poetry install
   ```
