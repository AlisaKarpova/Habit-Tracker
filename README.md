# Трекер привычек
Проект по созданию трекера для отслеживания полезных привычек и формирования новых жизненных установок. Он помогает систематизировать повседневные дела, фиксируя выполнение желаемых действий и мотивируя пользователей продолжать и достигать поставленных целей постепенно и эффективно.
