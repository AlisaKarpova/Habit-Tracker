# Трекер привычек
Проект по созданию трекера для отслеживания полезных привычек и формирования новых жизненных установок. Он помогает систематизировать повседневные дела, фиксируя выполнение желаемых действий и мотивируя пользователей продолжать и достигать поставленных целей постепенно и эффективно.

## Описание

Трекер привычек — это инструмент, который помогает пользователям отслеживать свои привычки и достижения. Он позволяет:
- Создавать и отслеживать привычки
- Устанавливать цели и отслеживать прогресс
- Анализировать данные и получать отчёты

## Установка

Для установки трекера привычек выполните следующие шаги:

1. Клонируйте репозиторий:
   ```
     git clone https://github.com/AlisaKarpova/Habit-Tracker.git
   ```
   
2. Установите зависимости:
   ```
   poetry install
   ```
   
## Примеры использования API

1. Запустить приложение:
```
uvicorn src.habit_tracker.models.fastapi_model:app --reload
```
2. Получить приветственное сообщение:
 - запрос curl:
```
curl -X 'GET' \
  'http://localhost:8000/' \
  -H 'accept: application/json'
```
- пример вывода:
```
{
  "message": "Привет! Расскажи о привычках, которые ты хочешь соблюдать",
  "motivational_quote": "Если ты можешь мечтать об этом, ты можешь это сделать."
}
```
3. Добавить пользователя:
 - запрос curl:
```
curl -X 'POST' \
  'http://localhost:8000/users/?name=%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80' \
  -H 'accept: application/json' \
  -d ''
```
- пример вывода:
```
{
  "user_id": "ff59ec46-b27b-4212-9003-407c482cb225",
  "message": "Пользователь Александр зарегистрирован. Скопируй и сохрани id. Он еще пригодится"
}
```
4. Добавить привычку:
 - запрос curl:
```
curl -X 'POST' \
  'http://localhost:8000/users/ff59ec46-b27b-4212-9003-407c482cb225/habits/?habits_name=20%20%D0%BE%D1%82%D0%B6%D0%B8%D0%BC%D0%B0%D0%BD%D0%B8%D0%B9%20%D0%B2%20%D0%B4%D0%B5%D0%BD%D1%8C&freq=%D0%B5%D0%B6%D0%B5%D0%B4%D0%BD%D0%B5%D0%B2%D0%BD%D0%BE&start_day=01-11-2025&end_day=30-11-2025' \
  -H 'accept: application/json' \
  -d ''
```
- пример вывода:
```
{
  "message": "Новая привычка '20 отжиманий в день' создана для пользователя Александр"
}
```
5. Если это необходимо, удалить привычку можно так:
- запрос curl:
```
curl -X 'DELETE' \
  'http://localhost:8000/users/ff59ec46-b27b-4212-9003-407c482cb225/habits/20%20%D0%BE%D1%82%D0%B6%D0%B8%D0%BC%D0%B0%D0%BD%D0%B8%D0%B9%20%D0%B2%20%D0%B4%D0%B5%D0%BD%D1%8C' \
  -H 'accept: application/json'
```
- пример вывода:
```
{
  "message": "Новая привычка '20 отжиманий в день' удалена для пользователя Александр"
}
```
6. Зафиксировать выполнение привычки:
- запрос curl:
```
curl -X 'POST' \
  'http://localhost:8000/habits/20%20%D0%BE%D1%82%D0%B6%D0%B8%D0%BC%D0%B0%D0%BD%D0%B8%D0%B9%20%D0%B2%20%D0%B4%D0%B5%D0%BD%D1%8C/mark/05-11-2025' \
  -H 'accept: application/json' \
  -d ''
```
- пример вывода:
```
{
  "message": "Привычка '20 отжиманий в день' зафиксирована как выполненная на 05-11-2025"
}
```
7. Проверить выполнение привычки в конкретную дату:
- запрос curl:
```
curl -X 'GET' \
  'http://localhost:8000/habits/20%20%D0%BE%D1%82%D0%B6%D0%B8%D0%BC%D0%B0%D0%BD%D0%B8%D0%B9%20%D0%B2%20%D0%B4%D0%B5%D0%BD%D1%8C/check/05-11-2025' \
  -H 'accept: application/json'
```
- пример вывода:
```
false
```
8. Рассчитать процент выполнения привычки:
- запрос curl:
```
curl -X 'GET' \
  'http://localhost:8000/habits/20%20%D0%BE%D1%82%D0%B6%D0%B8%D0%BC%D0%B0%D0%BD%D0%B8%D0%B9%20%D0%B2%20%D0%B4%D0%B5%D0%BD%D1%8C/rate' \
  -H 'accept: application/json'
```
- пример вывода:
```
{
  "message": "Процент выполнения привычки '20 отжиманий в день' составляет 3.33%"
}
```
9. Создать запись о выполнении привычки:
- запрос curl:
```
curl -X 'POST' \
  'http://localhost:8000/habits/20%20%D0%BE%D1%82%D0%B6%D0%B8%D0%BC%D0%B0%D0%BD%D0%B8%D0%B9%20%D0%B2%20%D0%B4%D0%B5%D0%BD%D1%8C/records/?day=01-11-2025' \
  -H 'accept: application/json' \
  -d ''
```
- пример вывода:
```
{
  "message": "Запись о выполнении привычки '20 отжиманий в день' создана успешно. Скопируй и сохрани id, он еще пригодится",
  "record_id": "d38eb37e-f9a4-4f08-aeb2-228b44d70083",
  "date": "01-11-2025"
}
```
10. Заполнить информацию о настроении:
- запрос curl:
```
curl -X 'POST' \
  'http://localhost:8000/habits/20%20%D0%BE%D1%82%D0%B6%D0%B8%D0%BC%D0%B0%D0%BD%D0%B8%D0%B9%20%D0%B2%20%D0%B4%D0%B5%D0%BD%D1%8C/records/d38eb37e-f9a4-4f08-aeb2-228b44d70083/mood?mood=%D1%82%D1%8F%D0%B6%D0%BA%D0%BE' \
  -H 'accept: application/json' \
  -d ''
```
- пример вывода:
```
{
  "message": "Настроение в записи 'd38eb37e-f9a4-4f08-aeb2-228b44d70083' обновлено"
}
```
11. Заполнить дополнительную информацию о выполнении привычки:
- запрос curl:
```
curl -X 'POST' \
  'http://localhost:8000/habits/20%20%D0%BE%D1%82%D0%B6%D0%B8%D0%BC%D0%B0%D0%BD%D0%B8%D0%B9%20%D0%B2%20%D0%B4%D0%B5%D0%BD%D1%8C/records/d38eb37e-f9a4-4f08-aeb2-228b44d70083/notes?notes=%D0%BE%D1%82%D0%B6%D0%B0%D0%BB%D1%81%D1%8F%20%D1%82%D0%BE%D0%BB%D1%8C%D0%BA%D0%BE%2012%20%D1%80%D0%B0%D0%B7' \
  -H 'accept: application/json' \
  -d ''
```
- пример вывода:
```
{
  "message": "Заметки в записи 'd38eb37e-f9a4-4f08-aeb2-228b44d70083' обновлены"
}
```
